# Prototype (static)

Лёгкий статический прототип (без бэкенда).

## Как открыть
1. Распакуйте архив.
2. Откройте файл `index.html` двойным кликом — он откроется в браузере.
3. Навигация по разделам — через верхнее меню.

## Что внутри
- Светлая/тёмная тема, зелёный акцент.
- Разделы: Каталог, Карточка, Вход, Профиль, Админ-панель.
- Верстка адаптивная (мобильная/десктоп).

## Структура
```
index.html
styles.css
app.js
assets/placeholder.svg
```

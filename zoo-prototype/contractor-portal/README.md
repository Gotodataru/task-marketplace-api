# Contractor Portal — Prototype (static)

Прототип личного кабинета исполнителя с выбором заявок, доходами, сообщениями и расписанием.
Пастельная палитра (зелёный акцент), светлая/тёмная тема. Без бэкенда — работает локально.

## Как запустить
1. Распакуйте архив.
2. Откройте `index.html` двойным кликом в браузере.
3. Разделы в верхнем меню. Заявки сохраняются в `localStorage` (можно тестировать приёмы/завершение).

## Разделы
- Главная: сводка, быстрые настройки, доступные заявки.
- Заявки: список доступных заявок, взять/скрыть.
- Мои: активные и завершённые заявки, завершить/отменить.
- Доходы: статистика и история.
- Расписание: заглушка с подключением календаря.
- Сообщения: заглушка для диалогов.
- Настройки: профиль и предпочтения.

## Стек
- Чистый HTML/CSS/JS, hash-router, localStorage.
